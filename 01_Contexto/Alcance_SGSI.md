# Alcance del Sistema de Gestión de Seguridad de la Información (SGSI)

## 1. Organización
CloudCore SaaS S.A.

## 2. Descripción del servicio
CloudCore SaaS S.A. proporciona una plataforma SaaS de gestión empresarial que incluye CRM y facturación electrónica para PYMES en Latinoamérica.

## 3. Alcance del SGSI
El SGSI cubre la protección de la información relacionada con el desarrollo, operación y soporte de la plataforma SaaS.

Incluye:
- Aplicación web
- API pública
- Infraestructura en la nube
- Procesos de desarrollo
- Gestión de accesos
- Soporte técnico

## 4. Activos incluidos
- Código fuente
- Bases de datos
- Infraestructura cloud
- Información de clientes

## 5. Ubicación
Operación remota + servicios en la nube.

## 6. Exclusiones
- Dispositivos de clientes
- Sistemas externos no gestionados directamente
